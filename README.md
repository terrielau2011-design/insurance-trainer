# 🛡 保險銷售視覺化互動培訓系統

**Insurance Sales Visualisation Training Tool v1.0**

## 線上預覽

👉 [https://terrielau2011-design.github.io/insurance-trainer/](https://terrielau2011-design.github.io/insurance-trainer/)

## 功能概覽

| 區塊 | 功能 |
|------|------|
| **A — The Hook** | 產品選擇、三大銷售亮點即時展示 |
| **B — The Presentation** | 場景一：儲蓄險（折扣/預預繳/權益牆）；場景二：保費融資（LTV/雙極限回報）；跨產品對比 |
| **C — The Close** | 一鍵生成客戶投保明白備忘錄、複製 WhatsApp、列印 PDF |

## 技術架構

- 純前端 HTML/CSS/JS，無需後端
- Chart.js 圖表引擎
- 響應式 CSS Grid 佈局
- 產品數據以靜態 JSON 內嵌於 `data.js`

## 自訂數據

修改 `data.js` 即可替換產品精算數據、亮點話術、權益門檻、銀行設定等。
